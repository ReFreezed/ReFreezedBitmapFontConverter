ReFreezed Bitmap Font converter
Developed by Marcus 'ReFreezed' Thunström

This universal version requires LÖVE 11.3 to be installed.

Download LÖVE: https://love2d.org/
Old versions:  https://github.com/love2d/love/releases

Use the program like this:

$ path/to/love RbmfConverter.love inputPath1 [inputPath2 ...] [options]
